So what is Djenga?

Great question. Djenga is an experimental webpage built by three Missouri Western State University students to better help them better understand Python. This site includes a blog built to demonstrate a journey of the efforts we went through to learn Django, a popular web framework for "perfectionists with deadlines" as deemed so by its creators. It also exists to show off our new skills as Python programmers.


The Django web framework is an incredibly-intuitive way to create websites. It works through the philosophy that every page contained in your website that isn't static should be considered an independent application. Django also provides integration for Jinja, a powerful Python templating-scheme that allows users to dynamically display their content. Check out our blog for more information on all things related to Python and Django, and also feel free to check out our contact page for additional information about this project's contributors and to gain a better overall understanding of who we are as developers.
